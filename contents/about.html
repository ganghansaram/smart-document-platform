KF-21 History WebBook 소개

KF-21 History WebBook은 폐쇄망(에어갭) 환경에서 사용 가능한 기술문서 통합 열람 플랫폼입니다. 3단계 접근 권한(열람자/편집자/관리자)을 지원하며, 문서 열람·검색·AI 챗봇 질문 등의 기능을 제공합니다.

항목내용
버전v5.5
동작 환경Chrome, Edge, Firefox (최신 버전 권장)
접속 방법브라우저에서 서버 주소 입력 후 로그인

주요 기능

문서 탐색

  • 트리 메뉴: 좌측 패널에서 계층적 메뉴를 통해 문서를 탐색할 수 있습니다.
  • 섹션 목차 (On this page): 우측 패널에 현재 문서의 제목 목차가 자동 생성됩니다. 클릭하면 해당 섹션으로 이동합니다.
  • 브레드크럼: 상단에 현재 문서의 메뉴 경로가 표시됩니다.
  • 패널 토글: 좌측·우측 패널을 숨기거나 표시하여 본문에 집중할 수 있습니다.

검색

  • 상단 검색창(Ctrl+K 또는 /)에 키워드를 입력하면 전체 문서에서 관련 내용을 검색합니다.
  • 키워드 매칭과 의미 기반 검색을 결합한 하이브리드 검색을 사용하여, 동의어나 유사 표현도 찾아줍니다.
  • 검색 결과 클릭 시 해당 문서의 정확한 섹션으로 바로 이동합니다.
  • 검색 결과 상단에 항공 용어집 일치 항목도 함께 표시됩니다.

AI 챗봇

  • 우하단 챗봇 아이콘을 클릭하여 문서 내용에 대해 자연어로 질문할 수 있습니다.
  • 시스템이 자동으로 관련 문서를 검색한 후, 해당 내용을 기반으로 한국어 답변을 생성합니다.
  • 멀티턴 대화: 후속 질문의 맥락을 유지하여 연속적인 대화가 가능합니다.
  • 빠른 질문 버튼: 현재 페이지의 요약, 핵심 내용, 쉬운 설명을 바로 요청할 수 있습니다.
  • 답변에 포함된 참고 문서 링크를 클릭하면 해당 섹션으로 이동합니다.

참고: AI 챗봇은 문서에 포함된 내용만을 기반으로 답변합니다. 문서에 없는 정보는 답변하지 않습니다.

북마크

  • 문서의 제목(h1~h4) 위에 마우스를 올리면 ☆ 아이콘이 나타납니다.
  • 클릭하면 ★로 변경되어 북마크에 저장됩니다.
  • 상단 Bookmarks 메뉴에서 저장된 북마크 목록을 확인하고, 클릭하면 해당 위치로 바로 이동합니다.
  • 다른 문서에 저장한 북마크도 클릭 한 번으로 이동됩니다.

다크/라이트 모드

  • 상단 헤더의 해/달 아이콘을 클릭하여 테마를 전환할 수 있습니다.
  • 선택한 테마는 브라우저에 저장되어 다음 방문 시에도 유지됩니다.
  • 인쇄 시에는 자동으로 라이트 모드가 적용됩니다.

항공 용어집

  • 좌측 메뉴의 용어집 항목에서 26,000+ 항공 용어를 검색·탐색할 수 있습니다.
  • A-Z 알파벳 필터, 실시간 검색(약어·영문·한국어)을 지원합니다.
  • 문서 본문에서 용어집에 등록된 약어에는 점선 밑줄이 자동 표시됩니다. 클릭하면 풀네임과 한국어 해설 팝업이 나타납니다.

그림/표 참조 팝업

  • 본문에서 "Figure 1", "표 2" 같은 참조 텍스트(파란색 점선 밑줄)를 클릭하면 해당 그림이나 표를 팝업으로 바로 확인할 수 있습니다.
  • 팝업에서 "원본 위치로 이동" 버튼을 클릭하면 해당 그림/표 위치로 스크롤됩니다.

설계 원칙

  • 폐쇄망 완전 독립: 외부 인터넷 연결 없이 모든 기능이 동작합니다.
  • 역할 기반 접근: 열람자·편집자·관리자 3단계 권한으로 기능 범위를 제어합니다.
  • 외부 의존성 Zero: CDN, 외부 API, 프레임워크를 사용하지 않습니다.
  • 설치 간편: 폴더 복사만으로 서비스를 시작할 수 있습니다.
